inside combo

This is another basic combobox. The events logged on the far right are events mounted TO THE ORIGINAL SELECT BOX.

UFFDD triggers some but not all events so that original events take place: namely the focus, change and blur events.


width control

Sizing defaults to autosize. UFFDD sets the exact same width as the master select, unless the original is below the "minWidth" option, or manualWidth is set.
