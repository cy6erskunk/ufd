basic combo

The combobox below is a very basic example of using UFD.


inside combo

This is another basic combobox. The events logged on the far right are events mounted TO THE ORIGINAL SELECT BOX.

UFD triggers some but not all events so that original events take place: namely the focus, change and blur events.


width control

Sizing defaults to autosize. UFD sets the exact same width as the master select, unless the original is below the "minWidth" option, or manualWidth is set.




Free text

The combobox below allows freetext, submitting the text back using the original selects' name. All events are still propagated back to master (including attempt to set if matched) even though it has been renamed (suffixed).
